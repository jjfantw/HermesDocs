💡 核心概念

Option 鍵 = Mac 的「側門」

Mac 上有個按鍵,「平時我們很難留意它,但是很多突然的選項都藏在它的後面」 — 影片原話。

Option 這個英文單字本身就是「另一種選擇」,所以 macOS 很多地方只要按住 Option 再點/拖,就會打開另一個跟預設行為不同的選項。

這些「選項」可以歸納成 三層作用:反向、揭示、增強。

🔑 Option 的三層作用

第一層 反向操作(另一種選擇)
核心邏輯:預設操作的反向版本。

0:03 正常按 = 輸入等號;按 ⌥+= 輸入不等號 ≠
⌥+Click
0:08 正常點 App 圖標 = 打開應用;按 ⌥+ 點 = 隱藏應用
⌥+Rotate
0:15 預覽圖片/影片時,預設只能向左旋轉;按 ⌥ 後 = 向右旋轉
第二層 揭示隱藏資訊(預設看不到的細節)
核心邏輯:macOS 平時把一些進階資訊藏起來,按 Option 可以臨時叫出來看。
⌥+Click WiFi
0:40 點 WiFi 圖標只看到基本連線;按 ⌥+ 點 = 顯示 IP 地址、速率、路由器、噪聲等更多網路資訊

0:46 Finder 選單列的「前往」預設看不到資源庫;按 ⌥+ 點「前往」就會把隱藏的「資源庫」顯示出來
⌥+Click 應用圖標
0:51 Dock 上對應用右鍵可以「退出」;按 ⌥+ 點 = 直接變成「強制退出」選項(跳過一般退出)

第三層 增強功能(把預設操作變強大)
核心邏輯:把日常操作從「基本款」升級到「Pro 版」。

🎯 學完可立刻用的 4 個場景

1. 拉 WiFi 故障排除 — 網路突然連不上? ⌥+ 點 WiFi 圖標,馬上看 IP/路由器/噪聲,不用開「網路」設定
2. 整理檔案想保留原檔 — 拖檔到新資料夾但不想搬走? ⌥+ 拖 = 複製一份過去
3. 調音量/亮度總覺得太大或太小 — ⇧+⌥ 同按音量/亮度鍵 = 4 倍細度,環境音特別好用
4. 應用當機 — Dock 上 ⌥+ 點應用圖標 = 跳過正常退出,直接強制結束

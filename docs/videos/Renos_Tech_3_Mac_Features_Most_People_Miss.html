重新排列選單列圖示

0:02

按住 Command 鍵，然後拖曳選單列（Menu Bar）上的圖示即可重新排列順序。

如果想移除某個圖示，直接將它拖離選單列即可完全刪除。這比進系統偏好設定調整快得多。

瞬間複製檔案

0:08

要快速複製檔案時，按住 Option 鍵，然後拖曳檔案到目標位置再放開。

這個方法比傳統的複製（Cmd+C）貼上（Cmd+V）流程快上許多，尤其適合在同一個資料夾內建立副本。

03

一鍵隱藏所有其他視窗

0:14

當桌面堆滿視窗時，按下 Command + Option + H，macOS 會立即隱藏所有非使用中應用程式的視窗，只保留當前正在使用的 App。

這比一個一個手動隱藏或使用 Mission Control 整理來得更快更乾淨。

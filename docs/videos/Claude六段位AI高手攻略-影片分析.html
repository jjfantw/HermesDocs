影片主張的關鍵洞察

  • 進化的軸心不是工具，而是你願意把哪一層工作交給 AI——任務 → 流程 → 能力 → SOP → 資料 → 營運 → 迭代。
  • 每一級都有對應的痛點：記憶、不可控、找嘸工具、SOP 卡住、權限複雜、除錯連鎖；這些痛點正是升級的入口。
  • L3 開始需要Skill + MCP 雙軌並進：沒有 MCP，Skill 只能在本機跑；沒有 Skill，MCP 只是空殼。
  • L5 起真實世界的摩擦才開始出現（Token、權限、錯誤排查），這也是 L6 Agent 系統設計的核心考量。
  • 最終的高手不是 L6，而是L6+：技術到位後，靠持續迭代拉開差距。
